Multi-antenna communications

OFDM-MIMO

A developing project area for multi-antenna OFDM simulation and receiver-side signal processing experiments.

Scope

Extend OFDM link-level simulation toward MIMO settings, with emphasis on channel representation, receiver processing, and empirical performance evaluation.

Methods

OFDM framing, antenna-domain channel modeling, detection and estimation baselines, BER and BLER analysis.

Planned Work

  • Define MIMO channel and resource-grid assumptions for reproducible experiments.
  • Implement receiver baselines for multi-antenna OFDM links.
  • Compare performance across channel conditions and antenna configurations.
